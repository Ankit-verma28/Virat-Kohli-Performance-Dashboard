# Virat Kohli Performance Dashboard

## Overview

This project is an interactive Power BI dashboard created to analyze Virat Kohli's batting performance across different formats. It provides insights into runs, batting average, strike rate, centuries, and overall career statistics through interactive visualizations.

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## What I Did

- Cleaned and transformed the dataset using Power Query.
- Created DAX measures for KPIs and calculations.
- Built interactive dashboards using charts, cards, and slicers.
- Analyzed batting performance using different visualizations.
